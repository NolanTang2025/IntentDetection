## 图片要求

- **尺寸**: 建议 800x600 像素或更高（宽高比约 4:3）
- **格式**: JPG, PNG 或 WebP
- **文件大小**: 建议每张图片不超过 500KB（可优化压缩）
- **背景**: 建议使用白色或浅色背景，突出商品主体
- **质量**: 清晰、光线充足，展示商品细节

## 注意事项

1. **文件名必须完全匹配**（区分大小写）：
   - `hoodie.jpg`
   - `sweater.jpg`
   - `pants.jpg`
   - `jacket.jpg`
   - `shirt.jpg`
   - `shoes.jpg`

2. 如果图片未上传或文件名不匹配，页面会自动显示 emoji 作为占位符

3. 支持的文件扩展名：
   - `.jpg` 或 `.jpeg`
   - `.png`
   - `.webp`

4. 图片会自动适应卡片尺寸，保持宽高比

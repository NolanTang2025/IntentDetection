# 用户意图分析可视化仪表板

专业的用户意图分析可视化工具，帮助独立站商家深入了解用户行为和意图。

## 📋 功能特点

### 1. 数据总览
- **用户聚类分布**：可视化展示所有用户聚类的分布情况
- **购买阶段分布**：了解用户处于哪个购买阶段（浏览/对比/决策）
- **价格偏好分布**：分析用户的价格敏感度
- **核心关注点**：识别用户最关心的产品特性

### 2. 用户聚类分析
- 深入分析每个用户聚类的特征
- 查看每个聚类的详细统计信息
- 了解不同用户群体的行为模式

### 3. 用户画像
- 详细的用户特征分析
- 核心兴趣、价格偏好、购买阶段
- 时间行为模式（活跃时段、浏览时长等）

### 4. 业务洞察
- 针对每个聚类的营销策略建议
- 产品推荐建议
- 转化优化建议
- 价格策略建议
- 内容策略建议

### 5. 产品偏好分析
- 产品关注度排名
- 不同聚类对产品的偏好
- 产品推荐策略

## 🚀 使用方法

### 方式一：直接打开（推荐）

1. 确保所有文件都在同一目录下：
   - `index.html`
   - `styles.css`
   - `dashboard.js`
   - `data.js`

2. 直接用浏览器打开 `index.html` 文件即可

### 方式二：本地服务器（推荐用于开发）

```bash
# 使用 Python 3
cd visualization_dashboard
python3 -m http.server 8000

# 然后在浏览器中访问
# http://localhost:8000
```

## 💡 使用建议

### 对于商家

1. **日常监控**：
   - 定期查看"总览"页面，了解整体用户意图分布
   - 关注"产品偏好"页面，了解热门产品

2. **营销策略制定**：
   - 查看"业务洞察"页面，获取针对性的营销建议
   - 根据用户聚类特征制定个性化营销策略

3. **产品优化**：
   - 查看"产品偏好"页面，了解用户最关注的产品
   - 根据用户画像优化产品推荐算法

4. **转化优化**：
   - 查看"用户聚类"页面，了解不同用户群体的特征
   - 根据转化优化建议优化网站和页面

### 对于数据分析师

- 可以导出数据文件进行进一步分析
- 所有数据都在 `data.js` 文件中，格式为 JSON
- 可以修改 `dashboard.js` 添加自定义分析

## 🔧 自定义

### 修改颜色主题

编辑 `styles.css` 文件，修改以下颜色变量：
- 主色调：`#667eea` 和 `#764ba2`
- 背景色：`#f8f9fa`
- 文字颜色：`#333`

### 添加新的图表

在 `dashboard.js` 中添加新的图表函数，参考现有的：
- `createPieChart()` - 饼图
- `createBarChart()` - 柱状图
- `createDoughnutChart()` - 环形图
- `createHorizontalBarChart()` - 水平柱状图

# 用户画像分析文件夹

本文件夹包含基于聚类结果的用户画像分析和业务指导建议。

## 文件说明

### 核心分析文件

1. **user_portrait_analyzer.py**
   - 用户画像分析主脚本
   - 功能：分析每个聚类的用户特征，生成业务洞察
   - 运行方式：`python3 user_portrait_analyzer.py`

### 分析结果文件

2. **user_portraits_detailed.json**
   - 详细的用户画像数据（JSON 格式）
   - 包含每个聚类的完整统计信息：
     - 用户规模、浏览时长、交互次数
     - 意图画像（核心兴趣、价格偏好、购买阶段等）
     - 产品偏好、行为模式、时间模式

3. **business_insights.json**
   - 业务洞察数据（JSON 格式）
   - 包含每个聚类的业务指导建议：
     - 用户细分名称
     - 关键特征
     - 营销策略、产品推荐、转化优化、价格策略、内容策略

4. **user_portrait_report.md**
   - 可读性报告（Markdown 格式）
   - 每个聚类的详细分析报告，包含：
     - 关键特征
     - 营销策略建议
     - 产品推荐建议
     - 转化优化建议
     - 价格策略建议
     - 内容策略建议

5. **business_insights_summary.csv**
   - 业务洞察汇总表（CSV 格式）
   - 便于在 Excel 中查看和分析
   - 包含所有聚类的关键信息汇总

6. **executive_summary.md**
   - 执行摘要（Markdown 格式）
   - 整体分析的核心发现和业务指导建议
   - 适合管理层快速了解分析结果

## 分析维度

### 用户画像维度

1. **基本统计**
   - 用户规模（独立用户数、片段数）
   - 浏览时长
   - 交互次数

2. **意图画像**
   - 核心兴趣（止鼾、睡眠质量、颈部疼痛、AI 技术等）
   - 价格偏好（高端、中端、经济型）
   - 购买阶段（浏览、对比、决策）
   - 主要吸引力（止鼾、睡眠质量、疼痛缓解等）
   - 关注点（功能、有效性、舒适度、价格等）

3. **产品偏好**
   - 关注的产品型号（Z6、A1、F1、H02、G1 等）
   - 产品偏好频率

4. **行为模式**
   - 高参与度用户比例
   - 快速浏览用户比例
   - 深度研究用户比例
   - 多会话用户比例

5. **时间模式**
   - 活跃时段
   - 活跃日期

### 业务指导维度

1. **营销策略**
   - 针对不同购买阶段的策略建议
   - 内容类型建议

2. **产品推荐**
   - 每个聚类的主推产品
   - 产品推荐逻辑

3. **转化优化**
   - 针对不同浏览行为的优化建议
   - 页面优化建议

4. **价格策略**
   - 针对不同价格敏感度的策略
   - 促销工具建议

5. **内容策略**
   - 针对核心关注点的内容建议
   - 时间优化建议

## 使用建议

### 快速了解
1. 先阅读 **executive_summary.md** 了解整体情况
2. 查看 **business_insights_summary.csv** 快速浏览各聚类

### 深入分析
1. 阅读 **user_portrait_report.md** 了解每个聚类的详细分析
2. 查看 **user_portraits_detailed.json** 获取详细数据
3. 查看 **business_insights.json** 获取业务洞察数据

### 业务应用
1. 根据各聚类的营销策略建议制定营销计划
2. 根据产品推荐建议优化产品推荐算法
3. 根据转化优化建议优化网站和页面
4. 根据价格策略建议制定促销策略
5. 根据内容策略建议制作营销内容

- **颈部疼痛缓解用户**：关注颈椎护理产品
- **睡眠质量改善用户**：综合睡眠解决方案
